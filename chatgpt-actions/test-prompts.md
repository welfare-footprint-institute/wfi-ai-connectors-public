# Test prompts for the WFI Canon Reader action

Use these prompts after importing the action into a Custom GPT.

This file is for testing the connector, not for teaching workshop participants.

## Basic retrieval test

Please retrieve `AI_USE.md` and `CANON_MAP.md` from the WFI Scientific Canon repository. Tell me which files you retrieved.

## Canon orientation test

Using the retrieved canon files, explain in plain language what the WFI Scientific Canon is and how an AI system should use it.

## Terminology test

Retrieve the terminology file and deprecated terms file. Then identify whether the following terms need review or replacement:

- Aggregation and Standardization
- Pain-Track GPT
- welfare score
- Biological Outcomes

## Module test

Retrieve the Module I, Module II, and Module III files. Then explain the difference between Circumstances, Biological Consequences, and Affective Experiences.

## Resolved decisions test

Retrieve the resolved methodological decisions file. Explain how it should be used in relation to the main canon files, and state that it should not override more specific current canon files unless the file itself says so.
