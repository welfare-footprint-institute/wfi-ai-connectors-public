# Future connectors

This folder is reserved for future connector specifications.

Possible future additions:

- Review/governance connector for canon feedback tools.
- MCP connector for Claude or other MCP-compatible tools.
- Connector for a WFI search endpoint.
- Connector for Atlas-ready structured outputs.

A future review/governance connector may expose files such as:

- `review/open_questions.md`
- `review/source_conflicts.md`
- `review/follow_up_items.md`
- `review/unresolved_methodological_issues.md`

Do not add these files to the default canon reader unless the GPT’s task explicitly requires canon governance or scientific review tracking.

Do not add experimental connector files here unless they are clearly labeled as draft.

Do not add private tokens, credentials, internal WFI material, workshop handouts, participant exercises, or app prototypes.
